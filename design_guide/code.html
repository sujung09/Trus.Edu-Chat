Could you explain the spacing rules again? Specifically how we should handle the 4px baseline grid in responsive layouts.

Tutor Sarah • 10:48 AM

Sure! Think of it as a multiplier. All margins and paddings should be increments of 4 (4, 8, 12, 16...). Here is a quick utility setup for CSS:

content_copy 
:root {
  --grid-unit: 4px;
  --spacing-sm: calc(var(--grid-unit) * 2); /* 8px */
  --spacing-md: calc(var(--grid-unit) * 4); /* 16px */
  --spacing-lg: calc(var(--grid-unit) * 6); /* 24px */
}
